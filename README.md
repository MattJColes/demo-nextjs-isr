# NextJS ISR Demo

This code goes a long with a demo I am giving of [Incremental Static Regeneration](https://nextjs.org/docs/basic-features/data-fetching#incremental-static-regeneration) from **NextJS**.

## Getting Started

- `yarn run dev`
